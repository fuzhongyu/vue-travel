# 项目安装的其他依赖

### stylus安装（css样式）
```
npm install stylus stylus-loader --save
```

### fastClick

处理移动端 click 事件 300 毫秒延迟，安装这个插件可以去除这个延迟
```
npm install fastclick --save
```

### awesome-swiper（轮播图）

```
npm install vue-awesome-swiper@2.6.7 --save
```
